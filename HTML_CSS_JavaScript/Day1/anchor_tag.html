So what does anchor tag do?
The anchor tag is used to create hyperlinks in HTML.
It allows users to click on a link and be taken to a different page or location.
by manipulating target tag you can control where the link opens (same tab, new tab, etc.)

The href attribute specifies the URL of the page the link goes to.

The target attribute specifies where to open the linked document.
